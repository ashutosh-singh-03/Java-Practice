# ATM Management System (Java)

A simple console-based ATM Management System built using Java and Object-Oriented Programming (OOP) concepts. The project simulates basic ATM functionalities through a menu-driven interface.

## Features

- Secure login with 4-digit PIN
- Maximum 3 login attempts
- Check account balance
- Deposit money
- Withdraw money
- View mini statement
- Change PIN
- Exit the ATM

## Technologies Used

- Java
- OOP Concepts
- ArrayList
- Scanner
- Packages

## How to Run

1. Clone the repository.
2. Open the project in your preferred Java IDE or VS Code.
3. Compile and run `App.java`.
4. Enter the PIN to access the ATM menu.
